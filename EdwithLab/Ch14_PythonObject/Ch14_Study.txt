Ch14 : 파이썬 오브젝트

Part1 : Object in Python

메소드 : 클래스 내의 함수
필드 / 속성 : 클래스 내의 변수


Part2 : Object & Class

클래스 : 템플릿
객체 : 인스턴스

x = list()
type(x) : x 가 어떤 클래스인지 알려준다.
dir(x) : x 가 속한 클래스의 모든 메소드 / 필드를 보여준다.

y = 'Hello, world!'
y 는 string 클래스이다.
즉, dir(y) 를 작성하면 string 클래스 내의 다양한 메소드 / 필드를 볼 수 있다


Part3 : Object Life Cycle

라이프 싸이클 : 생명주기
= 객체의 생성 - 삭제의 반복을 의미한다.

생성자 : 객체의 생성에 사용됨
> 객체를 정의할 때 자동으로 사용

소멸자 : 객체의 소멸에 사용됨
> 특정 변수를 덮어쓸 때 등에 자동으로 사용

Part4 : 상속

코드 재사용을 하기 위함임

부모 클래스로부터 상속을 받아, 자식 클래스가 부모 클래스의
메소드와 변수를 사용할 수 있다

개념 정리

클래스 : 형식 / 틀만 존재하는 것
속성 : 클래스 내의 변수
메소드 : 클래스 내의 함수
객체 : 클래스로 찍어낸 실제 존재하는 것
생성자와 소멸자 : 클래스를 생성하거나 삭제하는 함수
상속 : 다른 클래스의 기능을 물려받아 사용하는 것
